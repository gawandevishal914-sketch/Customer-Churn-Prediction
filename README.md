# Customer-Churn-Prediction
Machine Learning project that predicts telecom customer churn using classification algorithms like Decision Tree and Random Forest with data preprocessing, visualization, and CLI-based prediction system.
# Customer Churn Prediction using Machine Learning

## Overview

Customer churn prediction is an important business problem in the telecom industry. This project uses Machine Learning algorithms to predict whether a customer is likely to leave the telecom service based on historical customer behavior and usage patterns.

The project includes:

* Data preprocessing and visualization
* Feature encoding techniques
* Decision Tree and Random Forest models
* Handling imbalanced datasets
* CLI-based prediction system

## Objectives

* Predict customer churn accurately
* Understand customer retention strategies
* Learn business-oriented ML model development
* Improve classification model implementation skills

## Machine Learning Concepts Used

* Classification Algorithms
* Decision Tree Classifier
* Random Forest Classifier
* Feature Encoding
* Data Visualization
* Model Evaluation Metrics
* Imbalanced Data Handling

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Outcome

The model predicts whether a customer will:

* **Churn** (leave the telecom service)
* **Not Churn** (continue the service)

This project helps understand real-world business prediction systems and customer retention analysis using Machine Learning.
